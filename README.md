# TradeByte

> Stocks Trading Simulation WebApp

## Build Setup

Add your MongoDB URI, Google OAuth credentials and [Alpha Vantage Keys](https://www.alphavantage.co/) to the config.env file see config.env.example for details.

```bash
# install dependencies
npm install

# run in production
npm start

# run in development
# serve with hot reload
npm run dev
```
